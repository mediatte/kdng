## 모바일로 Claude Code 사용하기, 이제 집에서도 카페에서도 가능합니다

> macOS/Windows PC의 Claude Code를 iPhone/Android에서 SSH로 실행하는 방법을 단계별로 정리했습니다. 기술 진입 장벽 없이 쉽게 따라 할 수 있는 구성입니다.

---

## 🛠️ 필요한 구성 요소

| 항목 | 내용 |
|------|------|
| **주요 도구** | Tailscale(네트워크 연결), Termius(SSH 터미널) |
| **지원 환경** | macOS, Windows + iOS, iPadOS, Android |
| **난이도** | 초급 ~ 중급 |
| **소요 시간** | 약 15~20분 |
| **비용** | Tailscale(개인용 3명/100대 기기 무료), Termius(호스트 무제한 무료) |

> TIP: Tailscale은 WireGuard 기반의 Zero Trust VPN으로 NAT 환경에서도 자동 연결을 지원합니다. 포트포워딩이 필요 없어 초보자도 쉽게 사용할 수 있습니다.

---

## 1️⃣ Tailscale로 네트워크 연결하기

### 1.1 PC에서 Tailscale 설치
- **macOS**:
  - Homebrew: `brew install tailscale`
  - [공식 다운로드 페이지](https://tailscale.com/download/mac)에서 설치
- **Windows**: [공식 다운로드 페이지](https://tailscale.com/download/windows)에서 설치

### 1.2 모바일에서 Tailscale 설치
| 플랫폼 | 다운로드 링크 |
|--------|---------------|
| iOS | [App Store](https://apps.apple.com/ca/app/tailscale/id1475387142?mt=12) |
| Android | [Google Play](https://play.google.com/store/apps/details?id=com.tailscale.iphon) |

### 1.3 계정 설정 및 연결
1. [Tailscale 로그인 페이지](https://login.tailscale.com/start)에서 계정 생성
2. PC와 모바일에서 동일 계정으로 로그인 (Google, GitHub 등 SSO 지원)
3. 양쪽 기기가 Tailscale 네트워크에 표시되면 연결 성공

### 1.4 Tailscale IP 확인
```bash
# macOS/Linux에서
tailscale ip -4
# 예시 출력: 100.x.x.x (Tailscale 내부 IP)
```

## 2️⃣ PC에서 SSH 서버 활성화

### 2.1 macOS SSH 활성화
- **시스템 설정** → 일반 → 공유 → **원격 로그인** 체크
- 또는 터미널에서:
  ```bash
  sudo systemsetup -setremotelogin on
  ```

### 2.2 Windows SSH 활성화
1. 설정 → 앱 → 선택적 기능 → "OpenSSH 서버" 설치
2. PowerShell(관리자 권한):
   ```powershell
   Start-Service sshd
   Set-Service -Name sshd -StartupType 'Automatic'
   ```

## 3️⃣ Termius로 SSH 접속 설정

### 3.1 Termius 설치
| 플랫폼 | 다운로드 |
|--------|----------|
| iOS | [App Store](https://apps.apple.com/app/termius-ssh-client/id1435061311) |
| Android | [Google Play](https://play.google.com/store/apps/details?id=com.termius) |

### 3.2 호스트 추가
1. Termius 실행 → Hosts → **+ (새 호스트)**
2. 입력 항목:
   - **Alias**: My MacBook
   - **Hostname**: Tailscale IP (100.x.x.x)
   - **Port**: 22
   - **Username**: PC 사용자명
   - **Password/Key**: PC 비밀번호 또는 SSH 키

> SSH 키 사용 시 더 안전합니다. PC에서 `ssh-keygen`으로 키 생성 후 Termius에 등록하세요.

## 4️⃣ Claude Code 실행하기

### 4.1 기본 실행
```bash
# Claude Code 실행
claude

# 특정 디렉토리에서 실행
cd ~/projects && claude

# 대화 재개/새 대화
claude --continue
claude --new
```

### 4.2 프로젝트별 실행 예시
```bash
# Obsidian Vault에서 실행
cd ~/Local\ Obsidian_MBP/CMDSPACE_Local_MBP && claude

# 특정 프로젝트
cd ~/DEV/my-project && claude
```

### 4.3 유용한 단축 명령어
```bash
# ~/.zshrc 또는 ~/.bashrc에 추가
alias c="claude"
alias cc="claude --continue"
alias cmds="cd ~/Local\ Obsidian_MBP/CMDSPACE_Local_MBP && claude"
```

## 5️⃣ 고급 팁

### 5.1 tmux로 세션 유지
```bash
# 세션 생성
tmux new -s claude

# 세션 분리 (Ctrl+B → D)
# 세션 재연결
tmux attach -t claude
```

### 5.2 Tailscale 항상 연결 유지
- **모바일**:
  - iOS: 설정 → Tailscale → **Always On VPN** 활성화
  - Android: Tailscale 앱 → **Always-on VPN** 설정

## 🚨 자주 발생하는 문제와 해결법

| 문제 증상 | 해결 방법 |
|-----------|------------|
| SSH 연결 거부 | PC에서 SSH 서버 활성화 확인 |
| Tailscale 연결 실패 | 양쪽 기기 계정 일치 확인 |
| "operation not permitted" (iOS) | iPhone 설정 → Termius → **로컬 네트워크 권한** 활성화 |
| Termius에서 "Connecting" 상태 | Tailscale 연결/IP 재확인 |

---

## ✅ 실제 사용 시나리오

1. **카페에서 코드 수정**: 모바일 데이터로 Tailscale 연결 → Termius로 집 PC 접속 → `claude` 실행 → "버그 수정해줘" 요청
2. **침대에서 아이디어 정리**: iPad로 Termius 실행 → Obsidian Vault 경로 이동 → `claude` 실행 → 아이디어 문서화 요청
3. **이동 중 프로젝트 작업**: SSH 세션 유지(tmux) → 터미널 연결 끊겨도 작업 지속 가능
