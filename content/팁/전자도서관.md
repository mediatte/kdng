# 국내외 전자도서관 현황

## 국내 전자도서관

### 국립중앙도서관
- **링크**: [https://www.nl.go.kr/](https://www.nl.go.kr/)
- **특징**: 국가전자도서관 시스템 운영, 국내 주요 도서관 디지털 콘텐츠 통합 검색 서비스 제공
### 국회도서관
- **링크**: [https://www.nanet.go.kr/](https://www.nanet.go.kr/)
- **특징**: 입법 정보 및 사회과학 자료 중심, 구독형 전자책 서비스 (장기 열람증 필요)
### 서울도서관
- **링크**: [https://elib.seoul.go.kr/](https://elib.seoul.go.kr/)
- **특징**: 시민 대상 전자책·오디오북 서비스, 다양한 분야 자료 제공

### 경기도사이버도서관
- **링크**: [https://ebook.library.kr/](https://ebook.library.kr/)
- **특징**: 경기도민 대상 전자책 서비스, 웹/모바일 앱 지원

### 리브로피아
- **링크**: [구글 플레이](https://play.google.com/store/apps/details?id=eco.app.libropia.v2&hl=ko) | [앱스토어](https://apps.apple.com/nz/app/전국-도서관-전자책-리브로피아/id371073985)
- **특징**: 공공/대학 도서관 연계, 전자책 대출·좌석 정보 확인 앱

### 기타 주요 사이트
- 문화체육관광부 국가전자도서관: [https://www.dlibrary.go.kr/](https://www.dlibrary.go.kr/)
- 서울시교육청 전자도서관: [https://e-lib.sen.go.kr/main](https://e-lib.sen.go.kr/main)
- 교보문고 전자도서관: [https://ebook.kyobobook.co.kr/dig/cff/e-library](https://ebook.kyobobook.co.kr/dig/cff/e-library)
- Yes24 전자도서관: [https://www.yes24.com/notice/eLibrary/eLibrary.aspx](https://www.yes24.com/notice/eLibrary/eLibrary.aspx)

## 해외 전자도서관

### Z-Library
- **링크**: [https://z-lib.id/](https://z-lib.id/)
- **장점**: 다양한 무료 자료, 연구자들에게 인기
- **단점**: 저작권 문제, 불법 콘텐츠 가능성

### Library Genesis
- **링크**: [https://libgen.mx/](https://libgen.mx/)
- **장점**: 무료 학술 자료, 다양한 파일 형식
- **단점**: 법적 차단 가능성, 구식 인터페이스

### Scribd
- **링크**: [https://www.scribd.com/](https://www.scribd.com/)
- **장점**: 무제한 구독 서비스, 오프라인 지원
- **단점**: 유료 구독 필요, 지역 제한

### Google 도서
- **링크**: [https://books.google.co.kr/](https://books.google.co.kr/)
- **장점**: 고급 검색, 글로벌 접근성
- **단점**: 무료 콘텐츠 제한, 지역별 차이

### Project Gutenberg
- **링크**: [https://www.gutenberg.org/](https://www.gutenberg.org/)
- **장점**: 무료 고전 문학, 다기기 호환
- **단점**: 최신 서적 부재, 기능 제한적

### Internet Archive
- **링크**: [https://web.archive.org/](https://web.archive.org/)
- **장점**: 다양한 콘텐츠, Open Library 통합
- **단점**: 계정 필요, 복잡한 인터페이스

### PDF Drive
- **링크**: [https://www.pdfdrive.com/](https://www.pdfdrive.com/)
- **장점**: PDF 중심, 빠른 다운로드
- **단점**: 멀티미디어 부재, 광고 존재

### LibriVox
- **링크**: [https://librivox.org/](https://librivox.org/)
- **장점**: 무료 오디오북, 글로벌 접근성
- **단점**: 퍼블릭 도메인 제한, 음질 편차

### Open Library
- **링크**: [https://openlibrary.org/](https://openlibrary.org/)
- **장점**: 무료 대여, 간편한 인터페이스
- **단점**: 회원 가입 필요, 대여 기간 제한

### Sci-Hub
- **링크**: [https://www.sci-hub.se/](https://www.sci-hub.se/)
- **장점**: 무료 학술 논문, 간편 검색
- **단점**: 법적 논란, 서버 불안정

### Anna's Archive
- **링크**: [https://ko.annas-archive.org/](https://ko.annas-archive.org/)
- **장점**: 통합 검색, 오픈 아카이브
- **단점**: 복잡한 인터페이스, 중복 콘텐츠

### OverDrive
- **링크**: [https://www.overdrive.com/](https://www.overdrive.com/)
- **장점**: Libby 앱 지원, 디지털 북클럽
- **단점**: 구식 UI, 콘텐츠 제약

### Mobilism
- **링크**: [https://forum.mobilism.org/index.php](https://forum.mobilism.org/index.php)
- **장점**: 방대한 자료, 커뮤니티 기반
- **단점**: 저작권 문제, 보안 리스크

### DigiLibraries
- **링크**: [https://digilibraries.com/](https://digilibraries.com/)
- **장점**: 무료 장르별 도서, 단순 탐색
- **단점**: 광고 의존, 품질 편차

### Booknet
- **링크**: [https://booknet.com/](https://booknet.com/)
- **장점**: 작가-독자 상호작용, 연재 모델
- **단점**: 콘텐츠 편중, 개인정보 수집

### Pima County E-Library
- **링크**: [https://www.library.pima.gov/elibrary/](https://www.library.pima.gov/elibrary/)
- **장점**: 학습 도구 통합, 접근성
- **단점**: 도서관 카드 필요, 지역 제한

### BookBoon
- **링크**: [https://bookboon.com/](https://bookboon.com/)
- **장점**: 실용 학습 콘텐츠, 전문가 큐레이션
- **단점**: 유료 모델, 기능 불만 사례

### Bookyard
- **링크**: [https://vsrnvr.ac.in/nvr/Library/bookyaers.htm](https://vsrnvr.ac.in/nvr/Library/bookyaers.htm)
- **장점**: 다양한 형식, 넓은 장르
- **단점**: 저작권 문제, 품질 불명확

### Open Culture
- **링크**: [https://www.openculture.com/](https://www.openculture.com/)
- **장점**: 고품질 무료 콘텐츠, 다언어 지원
- **단점**: 복잡한 UI, 최신 자료 부족

### ManyBooks
- **링크**: [https://manybooks.net/](https://manybooks.net/)
- **장점**: 폭넓은 컬렉션, 사용자 친화적
- **단점**: 신간 제한, 유료 혼합
