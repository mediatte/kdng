




## 핵심 개선 사항  
1,096개 커밋으로 이루어진 대규모 업데이트가 공개됐다. 이번 버전은 개발 생산성 향상에 초점을 맞추면서 보안 패치와 메모리 누수 최적화까지 포함해 종합적인 개선을 이뤘다.  
## 주요 기능 개선  
### 텍스트 입력 편의성 강화  
기존 터미널 입력의 불편함을 해결하기 위해 `Shift+Enter`로 간편 개행을 지원한다. 작은 변화지만 실제 사용 편의성을 크게 높인 부분이다.  
### 에이전트 커스터마이징: Hooks 기능 추가  
`frontmatter`에 `PreToolUse`, `PostToolUse`, `Stop` 훅을 정의할 수 있게 됐다. 이를 통해 출력 형식 강제 적용이나 특정 작업 차단 등 워크플로우를 세밀하게 제어할 수 있다.  
### Skills 시스템 강화: Forked Context & Hot Reload  
- `fork`를 frontmatter에 추가하면 별도 컨텍스트에서 스킬을 실행해 연구나 탐색 작업에 최적화된다.  
- `~/.claude/skills` 또는 `.claude/skills` 폴더 수정 시 Hot Reload 기능으로 변경사항이 즉시 반영된다.  
- 에이전트 타입 지정 및 `/` 명령어로 호출이 가능해져 유연성이 크게 증가했다.  
### 다국어 지원 개선  
모델 응답 언어를 자유롭게 설정할 수 있게 되면서 갑작스러운 영어 출력 문제도 해결됐다. 글로벌 사용자를 고려한 실용적인 업데이트다.  
### 보안 및 편의 기능 강화  
와일드카드 툴 권한(`*` 패턴 매칭)을 지원하며, `/teleport` 명령어로 `claude.ai/code`로 세션을 빠르게 이동할 수 있다.  
## Claude Code의 가능성: Ethan Mollick 교수 사례  
Ethan Mollick 교수는 "한 달에 $1,000 벌 수 있는 완전 자동화된 웹 스타트업 구현"이라는 도전적인 과제를 제시했다. Claude Code는 다음과 같이 작업했다:  
1. 세 가지 객관식 질문으로 요구사항을 명확히 했다.  
2. 전문가용 프롬프트 500개 세트를 판매하는 비즈니스 모델을 제안했다.  
3. 1시간 14분 동안 추가 지시 없이도:  
   - 수백 개의 코드 파일과 프롬프트를 생성  
   - 단일 실행 파일로 배포 가능한 작동 웹사이트 완성  
   - 실제 결제 가능한 시스템 구현  
## 시사점  
이번 업데이트는 프로그래머뿐만 아니라 데이터 연구자, 디자이너 등 모든 창의적 작업자에게 유용한 도구를 제공한다. AI는 적절한 하네스(도구 체계)가 있을 때 지속적이고 중요한 작업을 수행할 수 있음을 입증했다. 현재 기능의 한계를 넘어 가까운 미래에 더 다양한 지식 작업에 적용될 전망이다.
